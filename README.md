# 商品采集助手插件 V1.0.3.7 · 规格值拆分与长规格修正版

基底：V1.0.3.6。

本版只修两个目标：

1. 点选一组规格值时，优先按子规格项拆分，避免“1 / 2 / 3”被合成“123”或一个大规格。
2. 保留长规格值，例如“猫咪/小型犬推荐（可裁剪-项围36cm）”，不再因为“推荐”、括号、斜杠被过滤。

保留：
- V1.0.3.5 的价格 / 库存 / SKU 图备注辅助带入逻辑；
- SKU 明细表可编辑；
- 图片候选池；
- 导出 ZIP 商品包，兼容主工具 V1.7.6.2。

不做：
- 不恢复 V1.0.4 / V1.0.5 的复杂入口；
- 不做 1688 全自动 SKU 解析；
- 不做淘宝 / 天猫深度 SKU 自动采集。
